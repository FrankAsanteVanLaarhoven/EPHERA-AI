# EPHERA vector masters

**Legal masters live here.** Do not treat AI-generated splash PNGs as final trademark assets.

|------|------|-----|

## Geometry (authoritative)

```
viewBox 0 0 96 96
bar height 14 · gap 7 · radius 7
top/bottom: x16 width 64
middle:     x16 width 48
```

Rebuild from these numbers in Figma/Illustrator; export EPS, PDF, PNG, WebP from **these paths**, not image traces.
