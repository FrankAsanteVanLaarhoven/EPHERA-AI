# EPHERA brand assets

All UI logo files are **transparent PNG** (backgrounds removed) with colours boosted for dark screens.

## Use by context

| Context | Asset |
|---------|--------|
| Home header | `wordmark-horizontal.png` via `EpheraLogo showWord` |
| Tab titles / small | `mark-ui.png` via `EpheraMark` |
| Splash | `wordmark-stacked.png` via `EpheraStackedLogo` |
| Voice orb / QR centre | `mark-ui.png` |
| Light / print | `logo-stacked-print.png` |
| Premium / metal | `logo-stacked-metal.png` |
| App icon / favicon | `icon.png` (mark on `#050B18` plate — required by stores) |

## Source kit (originals kept)

- `logo-*-glow.png` — original uploads (white plate; do not use in UI)
- Clean outputs: `logo-mark.png`, `logo-horizontal.png`, `logo-stacked.png`, etc.
